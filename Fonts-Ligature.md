# Copy ligatures

Use this tool from repository: [Ligaturizer](https://github.com/ToxicFrog/Ligaturizer) to copy ligatures of [Fira Code](https://github.com/tonsky/FiraCode) to any fonts you want

# Install fonts in Ubuntu

There are some ways to install fonts in Ubuntu:

* You can double-click on the font file (or select Open with Font Viewer in the right-click menu). Then click the Install Font button.
* You can place them in ~/.local/share/fonts directory
* To make them to be available system-wide, you need to copy them to /usr/local/share/fonts and reboot
